# Ahmed Gubran Cybersecurity Portfolio

A fully static and interactive cybersecurity portfolio built to showcase my cybersecurity background, SOC experience, penetration testing practice, certifications, projects, write-ups, and security reports.

## Features

* Dark cyber-themed responsive interface
* Interactive CyberBot assistant
* Linux-style command terminal
* Certificate and achievement gallery
* Cybersecurity projects and SOC experience sections
* Security reports and write-up library
* Social links for LinkedIn, GitHub, X, Medium, and TryHackMe

## Main Sections

* Professional Profile
* Technical Skills
* Certifications and Achievements
* SOC Experience
* Cybersecurity Projects
* Security Reports and Write-ups
* Contact Information

## Tech Stack

HTML, CSS, JavaScript, Static Assets, Cloudflare Pages / Workers

Most personal information, links, images, certifications, projects, and reports can be updated from:
